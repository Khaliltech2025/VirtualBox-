This tutorial demonstrates how to install and configure VirtualBox as well as displaying the capabilites of virtualization. <br/>

<h2>Environments and Technologies Used</h2>

Oracle VirtualBox Official Site

<h2>Operating System Used</h2>
Windows 11

<h2>List of Prerequisites</h2>

-Determine Web Browser of Choice.<br/>
-Navigate and Verify Official Site.<br/>
-Proceed Through the Installation Process<br/>
-Briefly Browse the Work Bench Dashboard<br/>
-Install the Extension Pack<br/>
-Read and Accept the Terms of Agreement <br/>
-Installation Progress Complete & Ready to Use for Lab Simulations<br/>

<p>
Hello and welcome to my IT tuorial that will be covering VirtualBox! VirtualBox is a powerful platform that allows an end user to harness the potential of Virtualization; gives the end user an alternative way to conduct labs, trouble shoot, create sandbox environments, and virtual machines rather than putting your personal computer (PC) and its resources at risk for things such as malware and viruses.
</p>

<img width="1365" height="767" alt="Step 1" src="https://github.com/user-attachments/assets/9e43581e-d8d6-4a0d-9ead-2a855340d8a1" />

<p> Step 1- Open up your preffered web browser of choice and click the first link to navigate to the offical site.</p>

<img width="1365" height="767" alt="Step 2" src="https://github.com/user-attachments/assets/b9a9aa75-42ff-4037-a8bc-5257a1e2bf8f" />
<p></p>

<p>Step 2- Verify that you're on the official download site (Prevents you from contracting malware or viruses that may be present on 3rd party sites).To verify, scroll to the bottom of the site and there should be a copyright symbol to verify you that is the official Oracle site. (ALWAYS INSTALL FROM THE SOURCE)!</p>

<img width="1365" height="767" alt="Step 3" src="https://github.com/user-attachments/assets/4bb6cf96-db3e-451d-85e2-4cd70bc36701" />

<p>Step 3- Scroll back up to the top of the screenand decide on which VirtualBox platform package is compatible for your operating System (OS). For example, I clicked on Windows Host since it is compatible with Windows 11. Next, it is time to determine which file path/ destination you'd like to download the application, then click "Save".</p>

<img width="1364" height="767" alt="Step 4" src="https://github.com/user-attachments/assets/2ef0e1cc-8b6f-4b63-bfec-cb68803061b7" />

<p>Step 4- Once the download is complete, make sure to double click on it so it will run the installation process. A user account control pop up message will ask if you'd like to make changes?  click "Yes".</p>

<img width="1365" height="767" alt="Step 5" src="https://github.com/user-attachments/assets/511ec59b-7575-4d3d-85cf-85c45b514e4f" />

<p>Step 5- Once you get to the welcome screen for the Oracle VIrtualBox installer click "Next". Make sure to read the End User License Agreement (E.U.L.A) & accept the terms of agreement. Then click "Next" you will see a screen that will let you configure the custom setup (I left the default settings as is, but you are more than welcome to change it to your liking.</p>

<img width="1365" height="767" alt="Step 6" src="https://github.com/user-attachments/assets/1c7c77ec-0e03-4f9f-992a-1bc1258f0323" />

<p>Step 6- After you make your configurations, you will receive a pop up warning. Click "Yes" (The warning essentially explains that it allows your virtual machine(s) to communicate to eachother on the network; crucial for Windows Server, Active Directory, Linux machines and etc.)Next, please make sure to click "Yes" for missing dependencies, you can also leave the custom setup as the default options toggled on (checkmark).</p>

<img width="1365" height="767" alt="Step 7" src="https://github.com/user-attachments/assets/01be6e0c-ea09-41a4-b488-13ee5f577e9e" />

<p>Step 7- Once you advance to the next pop up, click install.</p>

<img width="1365" height="767" alt="Step 8" src="https://github.com/user-attachments/assets/d63382ee-5163-4c29-9a6a-61acc4184879" />

<p>Step 8- Next, you will get another pop up saying that VirutalBox has been installed. Make sure to click "Finish"</p>

<img width="1365" height="767" alt="Step 9 " src="https://github.com/user-attachments/assets/25bbd1b4-74ae-415f-9eb7-4dd126ac93a5" />
<p>Step 9- Nice, you've officially launched VirtualBox and made it to the homepage which is known the "work bench".</p>

<img width="1365" height="767" alt="Step 10" src="https://github.com/user-attachments/assets/7ccd50e4-3610-4ebe-8665-2acbcbc33a1b" />
<p>Step 10- Go back to the official site and navigate to the VirtualBox extension pack. Make sure that the software version number matches with the VirtualBox software version number so it will be compatible when its time to perform projects and etc. Once you verify tat the verion numbers match, click "Accept & Download" (The extension pack is responsible for providing extra tools & functionality to make your VirtualBox lab session run more efficiently).</p>

<img width="1365" height="767" alt="Step 11" src="https://github.com/user-attachments/assets/c41fdc56-e607-44a6-bf4f-45f0f2602e6b" />

<p>Step 11- Once the extension package has been downloaded, click on the download button in the top right corner (above "All Bookmarks") and lcik on the file.</p>

<img width="1365" height="767" alt="Step 12" src="https://github.com/user-attachments/assets/b410379b-0537-4e03-8668-1c78fb338178" />

<p>Step 12- Once you click on the file, it should take you to the VirtualBox work bench screen and a pop up message will appear. Make sure that you press te "Install" / "Upgrade" button.</p>

<img width="1365" height="766" alt="Step 13" src="https://github.com/user-attachments/assets/fef0eaa0-4182-4fd3-8fce-90cb727fb096" />

<p>Step 13- Another pop up will appear. Please make sure to read over the terms of license agreement. Once you're finished, please make sure click "I Agree" to successfully add the extension pack to your work bench.</p>

<img width="1365" height="767" alt="Step 14" src="https://github.com/user-attachments/assets/f1818058-c97c-4a75-8b9d-426a0261b478" />

<p>Step 14- Congratulations, you've successfully downloaded Virtual Box! Now that you have this software on your device, you will be able to create virtual machines such as Windows, Linux Unbuntu, and Windows Server (in a lab setting). Windows server will be responsible for things such as Active Directory, Dynamic Host Configuaration Protocol(DHCP) and Domain Name System (DNS). Ultimately it will allow us to practice using these tools and learning the skills that would be necessary for working in a IT Support environment from te comfort of your own home for free!</p>
